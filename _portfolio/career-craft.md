Career Craft is an MCP agent that allows a user to upload or paste a resume, parses metadata (skills, roles, experience), and recommends suitable job openings.

Key features:
- Resume parsing and entity recognition (roles, skills, dates)
- Job recommendation engine that matches opportunities to parsed resume metadata
- Generative AI assistant to draft tailored cover letters and suggest resume edits
- Exportable, ATS-friendly resume formatting and cover letter templates

Tech highlights: Natural Language Processing, semantic search, prompt engineering, lightweight agent orchestration.
