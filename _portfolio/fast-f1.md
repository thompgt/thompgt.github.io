Fast F1 lets users visualize F1 championship battles and run what-if simulations across different points systems.

Highlights:
- Interactive championship leaderboards and per-race breakdowns
- What-if simulator to compare alternative points allocation schemes
- Visualizations for driver and team performance over seasons
- Exportable charts and CSV summaries for further analysis

Tech highlights: interactive plotting, simulation engines, and a lightweight web UI for scenario running.
